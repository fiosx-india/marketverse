# Guardian

## Purpose
Guardian is the internal monitoring and control system for MarketVerse.

## Responsibilities

- Monitor all modules
- Detect missing files
- Detect import errors
- Detect dependency issues
- Verify file connections
- Show Green / Yellow / Red status
- Generate popup alerts
- Report problems to the Main Dashboard
